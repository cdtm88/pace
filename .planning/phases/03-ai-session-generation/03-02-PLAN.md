## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user free-text ‚Üí AI prompt | goals/injuries are untrusted free text; must be data (XML-delimited), not instructions |
| AI output ‚Üí DB write | Claude JSON is untrusted; validated by Zod + safety gate before insert |
| client ‚Üí Server Action | readinessScore is the only client input; userId derives from cookie, never client |
| app ‚Üí Anthropic API | ANTHROPIC_API_KEY must stay server-side (never in client bundle) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-05 | Tampering | Prompt injection via goals/injuries (buildUserPrompt) | mitigate | User free-text wrapped in `<user_profile>`/`<injury_notes>` XML delimiters; SYSTEM_PROMPT is a static server constant and never receives user input (PROJECT.md ¬ßAI prompt safety) |
| T-03-01 | Tampering | AI output ‚Üí DB (GeneratedSessionSchema.safeParse) | mitigate | Action calls safeParse before insert; Zod-fail returns generic error, no insert |
| T-03-02 | Tampering | Unsafe session (validateSessionSafety) | mitigate | Action runs the safety gate after Zod, before insert (D-09 order, Pitfall 5) |
| T-03-03 | Denial of Service | Runaway AI spend (generationLimiter) | mitigate | `generationLimiter.limit(userId)` checked BEFORE the Anthropic call; blocked requests never reach the SDK |
| T-03-06 | Information Disclosure | ANTHROPIC_API_KEY in client bundle | mitigate | Imported from `@/env` in a `'use server'` module; no `NEXT_PUBLIC_` prefix (Phase 1 D-13) |
| T-03-07 | Information Disclosure | raw_json / technical error leakage to client | mitigate | `rawJson` stored server-only and never returned to the UI; Zod issues / safety reason logged server-side only; client gets generic messages |
| T-03-04 | Elevation of Privilege | Cross-user profile read | mitigate | `findUserProfileByUserId(userId)` uses cookie userId only; no client-supplied id |
| T-03-SC | Tampering | npm install `@anthropic-ai/sdk@0.104.1` | accept | RESEARCH.md Package Legitimacy Audit: official anthropics package, 23.5M downloads/wk, SUS-by-automation (release recency) only ‚Äî Approved, no [SLOP], no human checkpoint required |
